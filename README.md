# PulmoRehab Kit

An open-source starter toolkit for pulmonary rehabilitation education and simple rehabilitation workflow support.

## Overview
PulmoRehab Kit is a lightweight open-source project designed to support pulmonary rehabilitation education, patient tracking, and simple rehab-oriented workflows.

This project aims to provide practical, reusable resources for students, clinicians, researchers, and developers interested in pulmonary care and perioperative recovery.

## Why this project exists
Pulmonary rehabilitation often relies on fragmented educational materials, inconsistent tracking tools, and limited lightweight digital resources. This repository is intended to gradually grow into a practical starter toolkit that is open, reusable, and easy to improve.

## Current scope
This repository currently focuses on:
- rehabilitation tracking templates
- patient education materials
- simple starter scripts
- documentation for future development
## How to use
- review the starter templates in `templates/`
- adapt the rehab log CSV for your own use case
- run the Python summary script in `scripts/summary.py`
- review the roadmap in `docs/project-roadmap.md`
- see `docs/run-summary-script.md` for a basic script usage guide
## Project status
This project is in an early-stage maintenance and iteration phase. Recent updates include a basic usage guide, a starter field-definition document for the rehab log, and small improvements to the summary script.
